# 자동차 경주 게임

## 요구사항

- [x] 횟수(자동차가 몇번의 이동을 할 것인지)를 입력받는 기능
    - [x] 횟수만큼 게임을 진행하면 종료한다.
- [x] 자동차의 이름을 출력하는 기능
- [x] 자동차의 위치를 출력하는 기능


- [x] 자동차가 전진하는 기능
    - [x] 0에서 9 사이의 랜덤한 값을 생성하는 기능
    - [x] 랜덤한 값이 4 이상이면 전진한다.
- [x] 자동차의 이름을 부여하는 기능
    - [x] 쉼표(,)를 기준으로 이름을 구분한다.
    - [x] 이름은 5자 이하다.


- [x] 우승자를 알려주는 기능
    - [x] 우승자는 한명 이상일 수 있다.
